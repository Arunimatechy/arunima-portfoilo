## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Arunimatechy/arunima-portfoilo.git
```

Navigate to project

```bash
cd arunima-portfoilo
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
